## ğŸ”— Endpoints

### âš™ï¸ Generate Question Paper
- **Endpoint:** POST http://localhost:3000/api/question/
- **Description:** Generates a question paper based on total marks and difficulty percentages.
- **Example Request Body:**
  ```json
  {
    "totalMarks": 100,
    "easyPercentage": 30,
    "mediumPercentage": 40,
    "hardPercentage": 30
  }
  ```
- **Example Response:**
  ```json
  {
    "questionPaper": [
      // List of questions with details
    ],
    "totalMarks": 100
  }
  ```

### âš™ï¸ Generate Question Paper by Topic
- **Endpoint:** POST http://localhost:3000/api/question/topic
- **Description:** Generates a question paper based on total marks and topic percentages.
- **Example Request Body:**
  ```json
  {
    "totalMarks": 100,
    "difficultyPercentages": {
      "Easy": 30,
      "Medium": 40,
      "Hard": 30
    },
    "topicPercentages": {
      "Topic1": 20,
      "Topic2": 30,
      "Topic3": 50
    }
  }
  ```
- **Example Response:**
  ```json
  {
    "questionPaper": [
      // List of questions with details
    ],
    "totalMarks": 100
  }
